# Cores

Usamos CSS para alterar cores no nosso documento

## Tipos

* background-color (para caixas)
* color (para textos)
* borde-color (para-caixas)
* outros...

## Valores


Podemos definir os valores por 

* palavra-chave (blue, transparent)
* hexadecimal(#ffffff)
* funções rgb, rgba, hsl, hsla
